# @iooo_tttt 스레드 글쓰기 스타일 요약

> 전체 분석: `/mnt/c/dev/threads-data/analysis/writing-style.md`

## 핵심 규칙 (글 수정/작성 시 즉시 적용)

### 문장 구조
- **한 줄에 한 의미**. 문장이 아닌 절(clause) 단위로 줄바꿈
- 3~4줄 쓰고 빈 줄로 단락 구분 (숨쉬기)
- 긴 서술문 절대 금지. 호흡 짧고 리듬감 있게
- 3줄 이상 이어지는 문단 없음

### 톤
- 친근한 반말+존댓말 혼용: "~해요", "~거든요", "~하죠?"
- 딱딱한 격식체("~합니다", "~입니다"만) 금지
- 자기 비하 유머, 솔직한 감정 드러내기
- 겸손하면서도 전문적 ("1000쌤"다운 톤)

### 시그니처 표현
- "간딴하죠?", "암만 조타케도", "노션하세요!"
- "커밍쑤운 -!", "두둥-!", "스친님들"
- "진짜루..", "곰방", "촘", "맹글기"
- 구어체: "할라고", "만들라고", "알려드릴라니깐요"

### 이모지
- 거의 모든 게시물에 1~5개
- 자주 쓰는: 😂🤣😋🙌🏻👍🏻🥹👀✨🔥
- 피부색 포함 손 이모지 특징적
- 문장 끝 or 줄바꿈 후 단독 배치

### 구조 (팁/정보 공유형)
```
[짧은 훅 — 감정/숫자/질문/반전]

[구체적 방법/사례 — 넘버링/체크리스트]

[마무리 — CTA or 여운]
```

### 금기
- 긴 문단, 영어 전문용어 남발, 과장 광고 톤
- 해시태그 거의 안 씀
- "ㅎㅎ" 같은 어색한 웃음 (이모지로 대체)

### 체크리스트
- [ ] 첫 2줄에서 훅 걸리는가?
- [ ] 한 줄에 한 의미만?
- [ ] 3~4줄마다 빈 줄?
- [ ] 구어체로 자연스러운가?
- [ ] 200~300자 이내인가?
- [ ] 마지막에 행동 유도(CTA)?
