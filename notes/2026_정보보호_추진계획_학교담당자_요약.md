# 2026년 정보보호 추진 계획 — 학교 업무담당자 의무사항 요약

> **출처**: [붙임] 2026년 경기도구리남양주교육지원청 정보보호 추진 계획.hwpx
> **발행**: 경기도구리남양주교육지원청 감사과 (2026. 4.)
> **대상**: 각급 학교 업무담당자

---

## 📅 정기 의무 사항

### 매월 (월 1회)

- **사이버보안진단의 날** — 매월 **세 번째 수요일**, 모든 PC 보안진단 + 월별 중점점검 사항(별첨3) 실시
- **PC 개인정보 파일 점검** — Privacy-i로 매월 점검 및 보호조치(암호화·파기)
- **사이버보안진단 일지 작성** (별첨2 양식)

### 연 1회 (정기)

| 시기 | 항목 | 비고 |
|---|---|---|
| **2월** | 학교 업무담당자 정보보호 교육 이수 | G-ONE 원격 |
| **연중** | 전 직원 개인정보보호·정보보안 교육 | 각 연 1회 이상, **의무** |
| **상시 (연 1회 이상)** | 개인정보처리시스템 취약점 점검 | |
| **연 1회 이상** | 내부 관리계획 이행 실태 점검 | 접근권한·접속기록·암호화 |
| **6월·12월 (연 2회)** | 홈페이지 도메인(URL) 운영정보 현행화 | |
| **연 1회 정기** | PC 개인정보 파일 암호화·파기 결과 보고 | |

### 반기 1회

- 공공시스템(나이스, 처음학교로) 접근권한 변경·말소 점검 (해당 담당자)

### 4년 주기

- **학교 정보보호 감사** 수감 (구리남양주교육지원청 대상 43교 중 해당 시)
  - 미흡사항 조치 결과 **30일 이내** 보고 의무

## ⚡ 사고 발생 시 (의무 신고 기한)

| 사고 유형 | 기한 | 신고처 |
|---|---|---|
| **긴급 침해사고** | 인지 즉시 신고 | GECSC |
| **주요·일반 침해사고** | **24시간 이내** 신고 | 경기교육사이버위협 정보공유시스템 |
| **개인정보 유출 (1명 이상)** | **72시간 이내** 신고 | 각급학교 → 교육지원청 → 도교육청 |
| **개인정보 유출 — 정보주체 통지** | **72시간 이내** 통지 | 정보주체 직접 |
| **보안사고** | 즉시 학교관리자 보고 → 상급기관 신고 | |

### 개인정보보호위원회 직접 신고 대상
- 1천명 이상 유출
- 민감정보·고유식별정보 1건 이상 유출
- 외부로부터의 불법적인 접근에 의해 1건 이상 유출
- → **개인정보포털(privacy.go.kr)** 로 직접 신고

---

## 🔄 상시 준수 의무 (학교 보안관리 사항)

- 나이스·에듀파인 접속 단말기에서 카카오톡·메신저·밴드·SNS 사용 시 보안관리 준수
- 비공개 자료(개인정보·민감정보)는 **암호 설정** 후 업무용 메신저로 전송
- 「학교 인터넷 보안 가이드라인(교육부 2023.11)」 준수
- **생성형 AI에 개인정보·비공개 업무자료 입력 금지**
- 개인정보 취급자 **보안서약서 작성**
- 비업무사이트(게임·도박·P2P·상용메일·상용메신저) 차단
- 정보화사업 추진 시 → 교육지원청 감사과 경유 보안성 검토 요청

---

## 📌 기타 (해당 시)

- **개인정보파일 신규 운용** → 개인정보보호 종합지원시스템(intra.privacy.go.kr) 등록
- **개인정보 처리방침 개정** → 파일 정비 결과 반영해 홈페이지 게시
- **위탁사업** 발생 시 → 표준 계약서·점검표 활용, 수탁자 관리감독
- **신규·전입 직원** 발생 시 → 연 1회 이상 개인정보보호 교육 (의무)

---

## 🎯 핵심 체크포인트

1. **매월 셋째 수요일 = 사이버보안진단의 날** (절대 누락 금지)
2. **개인정보 유출 시 72시간 룰** (정보주체 통지 + 신고)
3. **별첨3 월별 중점점검표** 인쇄 후 매월 항목 따라가기
4. **연 1회 이상 전 직원 교육** 이수 확인 (의무, 미이수 시 법령 위반)
