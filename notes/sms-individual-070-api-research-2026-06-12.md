# 국내 SMS 발송 서비스 리서치 보고서
**주제: 개인(비사업자) + 070 인터넷전화 발신번호 + API + 자유로운 발송량**
(조사일: 2026-06-12, 공식 사이트 직접 확인 기반)

---

## 1. 결론 요약

- **1순위: 솔라피(SOLAPI, 구 CoolSMS)** — 개인 계정 가입 가능, **공식 FAQ에 "070번호(인터넷 전화번호)는 등록하여 사용하실 수 있습니다"라고 명시**된 유일한 서비스. REST API/SDK가 가장 개발자 친화적. 단, 개인 계정 일일 발송 한도는 기본 50건 → 최대 500건(무료 상향)이며 그 이상은 사업자 인증 필수.
- **2순위: 뿌리오(Ppurio)** — 개인 회원 가입 가능 + API 지원. 약관상 개인 1일 500통 제한이나, **"사전승인을 얻은 경우 발송량 제한 없이 발송 가능"한 예외 조항이 약관에 명시**된 점이 특징. 다만 070 등록 가능 여부는 공식 명시가 없음(본인 명의 가입증명원 제출 경로로 가능할 것으로 추정).
- **"발송량 제한 없음"은 개인 계정으로는 사실상 불가능.** 솔라피·뿌리오가 공통으로 "개인 1일 500건" 상한을 두고 있어 업계 공통 기준으로 보인다(스팸 방지 가이드라인 기반으로 추정). 대량 발송이 목적이라면 사업자 인증(개인사업자 포함)이 현실적인 해법.
- 다크호스로 **네이버 클라우드 SENS**: 발신번호 서류 인증 시 통신서비스 이용증명원의 서비스 종류에 "인터넷전화"가 공식 포함되어 있고, 개인 단위 일일 발송 한도 규정이 문서상 확인되지 않음. 단가도 SMS 9원으로 저렴. 단, 개인 회원 가입·실제 운영상 한도는 추가 확인 필요.

## 3. 서비스별 상세

### 솔라피 (SOLAPI) — 종합 1순위
- 발신번호 FAQ에 **"070번호(인터넷 전화번호)는 등록하여 사용하실 수 있습니다"** 명시, 통신사 발급 "통신서비스 이용증명원" 제출 필요. KT 인터넷전화는 biz.kt.com에서, LG U+ 인터넷전화는 고객센터(101) 문의 후 팩스 발급 안내까지 문서화되어 있음. 발급일 3개월 이내 서류만 인정, 가입신청서·요금납부내역서는 불인정.
- 일일 발송량: 개인 계정 기본 50건 → 최대 500건(무료, 사유 기재 필요·발송 이력 없으면 반려 가능). 500건 초과는 사업자 인증 필수. 사업자 기본 1,000건, 별도 조정 가능. 매일 오전 9시 초기화.
- 단가: SMS 18원 / LMS 45원 / MMS 110원, 월 발송량 따라 최대 57.69% 할인.
- 출처: solapi.com/pricing, solapi.com/guides/senderid-faq, solapi.com/guides/daily-limit/, solapi.zendesk.com(발신번호 서류 인증 360024293854, 등록 방법 360021357853)

### 뿌리오 (Ppurio)
- 개인 회원: 본인 휴대폰은 본인인증만으로, 본인 명의 일반번호는 통신사 가입증명원으로 등록(1~2영업일). 타인 명의는 개인회원 등록 불가(기업회원 전환 필요). 발신번호 최대 3개(추가는 기업회원만). "부가서비스로 받은 번호 및 선불폰은 발신번호 사용 불가" 명시 — 070 자체에 대한 가부 명시는 없음.
- 스팸정책 약관 제18조: "개인 이용자의 아이디 당 문자 전송량을 1일 500통으로 제한할 수 있습니다. 다만, 적법한 업무용 광고 발송, 동창회 연락 등… **사전승인을 얻은 경우 발송량 제한 없이 발송 가능하도록 예외를 인정할 수 있습니다**." — 개인 신분으로 한도 예외를 약관에 명시한 유일한 사례.
- 단가: SMS 8.8~27.5원 / LMS 27.3~50원 / MMS 55~230원(VAT 포함).
- 출처: ppurio.com/guide/sendCaller, ppurio.com/terms/spamPolicy, ppurio.com(요금), ppurio.com/send-api/guide

### 알리고 (Aligo)
- 가입: 사업자등록번호 없이 이름+휴대폰 본인인증으로 가입 가능해 보임(가입 페이지 기준). 단 회원분류 표기가 "사업자(개인/법인/단체)/공직선거"여서 순수 비사업자 개인의 지위는 모호 — 공식 확인 권장.
- 약관: 발신번호 사전등록 의무 + "스팸·피싱 방지를 위해 일일 발송량을 제한할 수 있다"고만 규정, 수치 비공개. 개인은 타인 명의 발신번호 등록 불가.
- 단가가 업계 최저권(SMS 8.4원)이며 REST API 제공(smsapi.html). 070 등록 가능 여부는 FAQ가 동적 페이지라 접근 불가 → 고객센터(02-511-4560) 확인 필요.
- 출처: smartsms.aligo.in, smartsms.aligo.in/join.html, smartsms.aligo.in/smsapi.html, m.aligo.in/smsterms/?terms=1

### 네이버 클라우드 SENS
- 발신번호 서류 인증: 통신서비스 이용증명원에 가입자 정보와 **"통신 서비스 종류(유선, 무선, 인터넷 전화 등)"** 가 기재되어야 하며 3개월 이내 발급분만 인정 — 즉 070 인터넷전화 증명원이 공식 수용 범주. 등록 신청 양식도 "소유주 또는 법인명"으로 개인 명의를 배제하지 않음.
- 개인 단위 일일 발송 한도 규정은 가이드 문서에서 확인되지 않음(= 명시적 한도 없음으로 단정은 금지). SMS 9원/건(공식 블로그 예시), 월 50건 무료.
- 출처: guide.ncloud-docs.com/docs/sens-callingno, api.ncloud-docs.com/docs/ai-application-service-sens-smsv2, medium.com/naver-cloud-platform (SENS 활용법)

### NHN Cloud SMS
- 개인/사업자 회원 모두 가입 가능(결제수단 차이: 계좌이체는 사업자만). 발신번호는 본인인증+서류 인증(통신서비스 이용증명원, 마스킹 없는 3개월 이내 발급분), "0N0 계열" 번호 등록 카테고리 존재.
- 발송 정책: **월 발송량 초기 5,000건 제한**(SMS/LMS/MMS/국제 합산), 조정은 고객센터 문의. 광고성 문자 규정(21시~08시 금지 등) 문서화.
- 출처: docs.nhncloud.com/ko/Notification/SMS/ko/sending-policy/, docs.nhncloud.com/ko/Notification/SMS/ko/console-guide/, docs.nhncloud.com/ko/nhncloud/ko/user-guide/

### 슈어엠 / 가비아 / 카페24 / 문자나라 (요약)
- **슈어엠**: 사업자용(슈어비즈)과 별도로 **사업자등록증 없는 개인용 웹 발송 포털**을 운영(검색 결과 기반). API는 기업용 중심으로 보여 "개인+API" 조합엔 부적합 추정. — surem.com, surem.co.kr
- **가비아**: API 문서 존재(message.gabia.com/api/documentation/). 신청 후 담당자 아웃콜·승인(2~3일) 필요. 개인 가입·요금은 공식 확인 불가.
- **카페24**: 쇼핑몰 부가서비스. 발신번호는 대표 운영자 명의 본인인증 또는 서류인증+전화 확인. 범용 API 용도로는 부적합. — support.cafe24.com
- **문자나라**: 요금표가 이미지로만 제공돼 추출 불가, 공식 API 문서도 미발견. 후보 우선순위 낮음. — munjanara.co.kr

## 4. 법적 제약 정리

**① 발신번호 사전등록제 (전기통신사업법 제84조의2)** — 모든 서비스 공통
- 본인 명의 **휴대폰**: 휴대폰 본인인증만으로 등록 (서류 불필요)
- 본인 명의 **070 인터넷전화 등 유선/일반번호**: **통신서비스 이용증명원** 제출 필수
  - 통신사별 명칭: 이용계약증명서(SKT/SKB), 통신가입증명원(KT), 서비스가입확인서·가입사실확인서(LG U+) 등
  - 공통 요건: **발급일 3개월 이내**, 마스킹(가림) 없는 원본, 가입자명·번호·발급일·발급 담당자 기재
  - 발급 경로: KT 인터넷전화 = biz.kt.com → [가입증명원 인쇄] / LG U+ 인터넷전화 = 고객센터 101 문의 후 팩스 수령 (솔라피 문서 기준)
  - 가입신청서·요금납부내역서는 증빙으로 불인정
- 즉, **070 명의자 본인이면 개인도 등록 가능**한 것이 제도의 기본 구조이고, 솔라피·SENS는 이를 공식 문서로 뒷받침. 단 일부 서비스(뿌리오)는 "부가서비스로 받은 번호"를 제외하므로, 별도 회선 계약 없이 부가서비스로 부여된 070은 거절될 수 있음.

**② 대량문자 전송자격인증제 (2024-06-01 시행)**
- 규제 대상은 **문자재판매사업자**(특수부가통신사업자)이지 개인 이용자가 아님. 개인이 직접 인증받을 일은 없음.
- 다만 이 제도와 스팸 방지 규제의 영향으로 각 사업자가 **개인 계정의 발송량을 자체 제한**: 솔라피(개인 최대 500건/일)와 뿌리오(개인 1일 500통)가 동일하게 500건 상한을 두는 것으로 보아 업계 공통 기준으로 추정됨(공통 기준의 공식 근거 문서는 확인 불가).
- 결론: **개인 신분으로 "무제한"은 법·업계 관행상 불가**. 1일 수백 건 이상이 상시 필요하면 개인사업자 등록 후 사업자 인증이 정석 경로.

## 5. 확인 불가 / 추정 항목 (정직 고지)

1. **뿌리오 070 등록 가부** — 공식 가이드에 070 명시 없음. "본인 일반번호=가입증명원" 경로로 가능할 것으로 추정 (부가서비스 번호 불가 조항과의 경계는 고객센터 1588-5412 확인 필요).
2. **알리고 비사업자 개인 가입 + 070 등록** — 가입 페이지상 사업자번호 필수 아님은 확인했으나, 회원분류 표기가 모호하고 발신번호 서류 FAQ는 동적 페이지라 내용 접근 불가. 고객센터 확인 필요.
3. **알리고 일일 발송 한도 수치** — 약관에 "제한할 수 있다"만 있고 수치 비공개.
4. **네이버 클라우드 개인 회원의 SENS 실제 이용 한도** — 한도 규정 문서를 찾지 못했을 뿐, "한도 없음"으로 단정 불가.
5. **NHN Cloud·가비아·슈어엠·문자나라·카페24의 SMS 건당 단가** — 동적 페이지/이미지/로그인 장벽으로 공식 확인 불가.
6. **개인 1일 500건 상한의 공식 출처** — 솔라피·뿌리오 공통 수치에서 업계 공통 가이드라인(방통위/KISA 스팸방지 가이드라인)으로 추정하나, 해당 가이드라인 원문은 이번 조사에서 직접 확인하지 못함.
7. **SENS SMS 9원** — 네이버클라우드 공식 Medium 포스트의 계산 예시 기준이며, 요금 페이지(동적 렌더링)에서 직접 재확인하지 못함.
