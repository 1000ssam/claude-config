# Wee Class Pro 2.0이란?

**Wee Class Pro 2.0**은 1000쌤(노션 공식 앰배서더)과 열음쌤(전문 상담 인플루언서)의 협력으로 제작된 **위클래스 상담 관리 노션 템플릿**입니다. 상담 일지 작성 한 번으로 나이스 업로드 서식과 월별 보고서가 자동 완성되어, 흩어진 상담 행정을 한 페이지로 통합합니다.

- **제작자**: 1000쌤 × 열음쌤
- **가격**: 49,000원 (60일 이용)
- **카테고리**: 교육 · 학교 · 상담
- **공급처**: 쌤동네 (학교 예산 구매 권장)

# 전문 상담 교사가 처한 현실

> "잦은 출장, 쌓여가는 서류 뭉치, 주기적 통계 보고…"

전문 상담 교사의 업무는 학생을 만나는 시간보다 **행정 처리 시간**이 더 길어지기 쉬운 구조입니다.

- 상담 일지는 한글 문서로 작성
- 나이스 업로드용 서식은 별도 정리
- 월별 통계는 엑셀로 다시 계산
- 분기별 보고서는 또 다른 파일

같은 데이터를 도구를 옮겨가며 **같은 손으로 다섯 번 적게 되는 구조**입니다. Wee Class Pro 2.0은 이 흐름을 한 번의 입력으로 끝내는 데 초점을 맞췄습니다.

# 핵심 기능

## 1. 상담일지 → 나이스 서식 자동 변환

상담일지를 노션에 작성하면 **나이스 업로드용 서식이 자동으로 채워집니다**. 같은 데이터를 두 번 입력할 필요가 없습니다.

## 2. 월별 보고서 자동 완성

매월 말 통계 보고를 위해 일일이 합산할 필요 없이, 누적된 상담 기록이 **월별 보고서로 자동 집계**됩니다. 보고서 작성 시간이 분 단위로 줄어듭니다.

## 3. 주간 활동 계획 관리

상담 일정·집단 프로그램·외부 출장 등 **주간 단위 활동 계획**을 한 페이지에서 관리합니다. 출장이 잦은 상담교사 업무 패턴에 맞춘 구조입니다.

## 4. 상담 대장 통합

학생별·사례별 상담 이력을 누적 관리하는 **상담 대장 DB**가 포함되어, 장기 사례 관리에 적합합니다.

## 5. v3.0 초대형 업데이트

> "더 개선된 사용자 여정과 레이아웃, 그리고 복사 한 번으로 완성되는 보고서까지!"

v3.0이 출시되었고 **2.0 구매자는 별도 결제 없이 v3.0 복제 방법을 쌤동네를 통해 전달**받을 수 있습니다.

- **나이스 업로드와 월별 보고서를 동시에 처리하시는 분** — 자동화로 시간 절약
