교실 빔프로젝터 강의용 PPT에서 사용자가 직접 내 산출물(ppt-deck)을 보정해 드러난 기준 (임진왜란 수업 덱, 2026-06-18):

- **글자 크기**: 틀(eyebrow 18pt·섹션제목 43pt·표지 72pt·질문제목 58pt)은 그대로 두되, **학생이 읽고 받아쓰는 본문/정답만 대폭 확대** — 카드 불릿 28pt, S 본문 25~36pt, 카드 핵심어(인물명) 48~54pt, 표 셀·목차 32~36pt. **본문 하한 ≈28pt.** 내 기본 1.2배 스케일(본문 14~18pt)은 뒷자리 가독성에 부족.
- **애니메이션**: 효과 1종(fade·onClick)으로 통일. **'질문 슬라이드 → 정리/답 슬라이드'의 답 내용에만** 걸어 클릭 시 공개. 정보 제시·사료·요약표는 안 가림(=답이 아니라 함께 읽는 자료). before/after는 이미 아는 쪽은 노출하고 새 정답만 가림. 전개과정은 ①⑥만 보이고 ②~⑤ 빈칸→클릭 공개.

**Why:** 발문→학생 작성→클릭 공개로 수업 호흡을 통제하고 답 미리보기를 차단하려는 의도. "장식이 아니라 정보 가독성"이 폰트 기준.
**How to apply:** 교육용 덱은 scale_fonts를 본문 28pt+로 상향, 정답 슬라이드 불릿에 fade-entrance(onClick) 자동 삽입. ⚠️ spec 재빌드는 수동 애니를 지우므로, 손작업 보존이 필요하면 재빌드 대신 기존 파일에 보강. [[project_ppt-deck-skill]]
