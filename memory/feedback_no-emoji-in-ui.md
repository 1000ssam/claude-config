---
name: UI 모킹에 이모지 사용 금지
description: 프리뷰/모킹 UI에서도 이모지를 네비게이션 아이콘으로 사용하지 않는다. 항상 SVG 아이콘 사용.
type: feedback
---

UI 프리뷰나 모킹 앱을 만들 때도 이모지(📊, 📋 등)를 아이콘 대용으로 사용하지 않는다.

**Why:** 이모지는 싸구려/비전문적으로 보이며, 실제 앱에서 절대 쓰이지 않는 요소를 프리뷰에 넣으면 디자인 평가 자체가 불가능해진다. 사용자가 이 점에 대해 강하게 불쾌감을 표현했음.

**How to apply:** 모킹 시에도 Lucide, Heroicons 등 SVG 아이콘을 사용하거나, 아이콘 없이 텍스트만으로 구성한다.
