사용자가 카피 일부 수정을 요청할 때 **그 부분만** 수정한다. 주변 문장이 길거나 어색해 보여도 임의로 압축·재작성하지 않는다.

**Why:** 2026-04-30 notiontalk-landing 작업 중, 사용자가 "한 명을 골라 자세히 살펴보세요. 이 병신같은 문구만 삭제해도 될듯?"이라고 마지막 한 문장만 지우라고 했는데, 직전 turn에서 "3줄로 만들기"를 위해 미리 다른 문장들까지 압축했음. 사용자가 화를 내며 "왜 죄다 지웠어 시팔" 반응. "만"이 명시되어 있었는데 무시함.

**How to apply:**
- 사용자가 "X만 지워" / "Y만 바꿔" / "Z만 넣어" 같이 **한정 표현(만)**을 쓰면 그 부분만 수정. 인접 문장은 손대지 않는다.
- 줄 수·길이 조정 같은 "구조적" 변경 요청도 사용자가 명시하지 않은 한 카피 자체를 줄이는 게 아니라 **컨테이너 폭·폰트 크기 같은 레이아웃**으로 먼저 풀어본다.
- 카피 압축이 자연스러워 보이는 상황이라도, 명시 요청 없으면 "압축안 제안"으로 먼저 묻고 적용은 사용자 OK 후에.
- 카피는 사용자의 톤·동기·맥락이 담긴 자산. AI가 임의로 다듬는 순간 사용자 자기 글이 아니게 됨.
