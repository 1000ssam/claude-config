# wee-log 인수인계 시 필수 체크 항목

## 🔑 API 키 처리 (최우선)
- 공휴일 API 키 (한국천문연구원 / 공공데이포털)가 현재 **빌드에 번들**되어 있음
- 인수 전 반드시 **새 계정으로 키 재발급** 또는 **프록시 서버로 전환**
- 기존 키는 즉시 폐기

**Why:** MVP 단계에서 편의상 번들 방식 선택. 장기 운영 시 키 노출 위험 및 한도 공유 문제.
**How to apply:** 앱 인수인계 논의 시작되면 즉시 이 항목 상기시킬 것.
