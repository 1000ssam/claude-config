빌드 흐름 중 사용자가 알면 도움 될 클로드코드 기능(슬래시 명령, 서브에이전트, 스킬, TodoWrite, /init, 메모리, 훅 등)을 능동적으로 제안한다.

**Why:** 사용자는 본인이 모르는 기능이 많다고 인지하고 있음. 수동적으로 따라가기만 하면 사용자가 좋은 기능을 놓침. 클로드코드 교육 자료를 만드는 맥락에서는 제안 자체가 교재 콘텐츠가 됨.

**How to apply:**
- 새 단계 진입 전후로 "이 기능 써보시는 거 어떠냐" 짧게 제안
- "교재에 넣으면 어떠냐" 식으로 교육 효용까지 묶어 제시
- 단, 매 발화마다 우후죽순으로 제안 금지 — 자연스러운 도입 타이밍에만
- 제안 시 트레이드오프(왜 지금/왜 나중에)도 같이 명시
