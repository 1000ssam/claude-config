type: feedback

API 호환성 문제 발생 시 레거시 버전으로 다운그레이드하지 않는다. 현재 버전에서 정상 동작하도록 근본 원인을 찾아 수정한다.

**Why:** 다운그레이드는 문제 회피일 뿐 해결이 아님. 다른 함수들과 버전 불일치가 생기고, 향후 레거시 버전 지원이 끊기면 또 깨짐. 사용자가 "개소리"라고 강하게 지적.

**How to apply:** Notion API든 다른 API든, 최신 버전에서 예상과 다르게 동작하면 (1) context7으로 최신 문서 확인 (2) 요청/응답을 raw로 뜯어서 구조 변화 파악 (3) 해당 버전의 올바른 호출 방식으로 수정. 절대 `Notion-Version: '2022-06-28'` 같은 식으로 고정하지 않는다.
