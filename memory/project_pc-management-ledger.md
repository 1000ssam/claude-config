## openpyxl 로드 주의사항
이 파일은 `_cell_styles` 인덱스 초과로 openpyxl 기본 로딩 시 IndexError 발생.
다음 패치 적용 후 로드:
```python
from openpyxl.utils.indexed_list import IndexedList
_orig = IndexedList.__getitem__
def _safe(self, i):
    try: return _orig(self, i)
    except IndexError:
        if isinstance(i, int) and len(self) > 0: return _orig(self, 0)
        raise
IndexedList.__getitem__ = _safe
```
또한 B열 관리번호 = 수식(`=LEFT(L,6)&"-"&TEXT(O,"0000")`). data_only=True 로드 시 None으로 보일 수 있으나 Excel에서 자동 재계산됨.
