description: 학교에서 사용하는 웹앱에는 '1000쌤' 대신 '김용천 선생님'으로 표기
type: feedback
---

학교에서 사용하는 웹앱(학생 대상)에서는 '1000쌤'이라는 이름을 쓰지 않는다. '김용천 선생님'으로 표기할 것.

**Why:** 1000쌤은 온라인 서브 페르소나이고, 학교에서는 사용하지 않는 이름이다.

**How to apply:** 학생 대상 교육용 웹앱(social-issue-research 등) 푸터, OG 이미지, 크레딧에 이름을 넣을 때 '김용천 선생님' 사용. 1000ssam 디자인 시스템 스킬의 기본 템플릿을 그대로 쓰지 말고 이름 부분은 교체.
