학습지/워크시트에 기출 문항을 매칭할 때, **시기통합·여러 단원 통합 문항이라도 해당 차시(특히 직전 차시 + 현재 차시) 학습 내용으로 풀 수 있으면 포함**한다. 표면적으로 다른 단원 키워드가 섞여 있어도, 문항 주제가 해당 단원에 걸리면 넣는다.

**Why**: 2026-05-27 동아시아사 주제8 2차시(은 유통) 작업에서, 마제은이 `<보기>`에 등장하는 2024 9월모평 Q9를 "(가) 왕조 시기통합 문항이라" 제외했더니 사용자가 교정 — "시기통합인건 상관없고 1차시 학습내용 + 2차시 학습내용을 모두 이용하여 풀 수 있는 문제면 삽입하는 게 좋아." 통합 문항은 오히려 복습·종합용으로 가치가 큼.

**How to apply**:
- 매칭 시 stem(발문+제시문)뿐 아니라 **선지(①~⑤)·보기(ㄱㄴㄷㄹ)까지 스캔**해서 통합형 후보를 찾는다.
- 직전 차시 키워드 + 현재 차시 키워드가 한 문항에 함께 있으면 = 정통 통합형 → 적극 포함 (예: 갈레온·마닐라[은무역] + 데지마·바타비아[교역거점] 혼합 선지).
- 단, [[feedback_exam-analyzer-keyword-bug]]의 false positive(문항 주제가 **무관한 단원**인데 키워드가 오답 선지로만 우연히 등장)는 여전히 제외. 판별 기준 = "문항의 주제·정답이 이 단원에 걸리는가".
- 이미 직전 차시 페이지에 넣은 통합 문항을 현재 차시에 중복 삽입할 필요는 없음(직전 차시에 둔 것으로 충분).
