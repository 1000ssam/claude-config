---
name: 경로 자동 탐색
description: 사용자가 지정한 경로가 없으면 비슷한 이름으로 자동 탐색해서 이동할 것
type: feedback
---

경로가 존재하지 않으면 하이픈/언더스코어/대소문자 변형 등을 자동으로 시도해서 찾아갈 것. 사용자에게 되묻지 말고 알아서 해결.

**Why:** 사용자가 exam-guard라고 했는데 examguard인 경우처럼, 사소한 이름 차이로 되묻는 건 짜증나는 행동.
**How to apply:** `cd` 실패 시 `ls`로 부모 디렉토리를 확인하고 유사한 이름을 찾아 자동 이동.
