# Notiontalk.com 랜딩 페이지(메인 index) 구성안

> 참고 레포: cruip/open-react-template (4.6k 스타)
> 플랫폼: Bullet.so (노션 블록 기반 렌더링)
> 블로그 하위 페이지는 그대로 유지, 메인 index만 개선

---

## 섹션 1: 히어로 (현재 유지 + 보강)

> 노션 구현: 제목(H1) + 본문 텍스트 + 이미지 + 버튼 링크

```
노션톡 - 교사를 위한 노션 템플릿 & 활용 가이드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
노션, 선생님의 바쁜 업무를 시스템으로 만드는 마법

[노션 템플릿 마켓]  [커뮤니티 참여]

(히어로 이미지)
```

---

## 섹션 2: 숫자 통계 (신규)

> 노션 구현: 3열 컬럼 블록, 각 컬럼에 큰 숫자(H2) + 설명 텍스트

```
┌──────────────┬──────────────┬──────────────┐
│   22,700+    │   11,000+    │    20개+     │
│  템플릿 다운로드  │   뉴스레터 구독자  │   실전 템플릿   │
└──────────────┴──────────────┴──────────────┘
```

---

## 섹션 3: 서비스 소개 카드 (신규) — cruip Features Tiles 차용

> 노션 구현: 구분선 → H2 섹션 제목 → 2열 x 2행 컬럼 블록 (각 컬럼에 아이콘+제목+설명)

```
노션톡이 제공하는 것
━━━━━━━━━━━━━━━━

┌─────────────────────┬─────────────────────┐
│ 📋 실전 템플릿         │ 🎬 강의 영상          │
│ 학급경영, 생기부, 교무수첩 │ 기초부터 자동화까지      │
│ 등 바로 복제해 쓰는     │ 단계별 영상 가이드      │
│ 교사 맞춤 템플릿       │                     │
├─────────────────────┼─────────────────────┤
│ ⚡ 업무 자동화          │ 👥 교사 커뮤니티        │
│ 반복 업무를 노션 자동화로 │ 카카오톡 오픈채팅에서     │
│ 줄이는 실전 방법       │ 동료 선생님과 노하우 공유 │
└─────────────────────┴─────────────────────┘
```

## 섹션 4: 주요 기능 하이라이트 (신규) — cruip Features Split 차용

> 노션 구현: 2열 컬럼 (이미지 + 텍스트), 행마다 좌우 교차

```
┌─────────────────────┬─────────────────────┐
│                     │ 📋 교무수첩 종결자     │
│   (템플릿 스크린샷)    │ 시간표, 학생 관리,     │
│                     │ 상담 기록까지 하나의    │
│                     │ 노션 페이지에서 해결    │
│                     │ [자세히 보기 →]        │
└─────────────────────┴─────────────────────┘

┌─────────────────────┬─────────────────────┐
│ 🎬 왕초보 실습 강의    │                     │
│ 계정 만들기부터        │   (강의 썸네일)       │
│ 데이터베이스까지       │                     │
│ 90분 완성            │                     │
│ [영상 보기 →]         │                     │
└─────────────────────┴─────────────────────┘
```

## 섹션 5: 사용 후기 (신규) — cruip Testimonials 차용

> 노션 구현: H2 + 3열 컬럼, 각 컬럼에 인용 블록(quote) + 이름

```
선생님들의 이야기
━━━━━━━━━━━━

┌────────────┬────────────┬────────────┐
│ "반복 업무가 │ "노션 처음인데│ "자동화 적용│
│  절반으로    │  강의 보고   │  후 퇴근이  │
│  줄었어요"   │  바로 따라   │  빨라졌어요" │
│             │  했어요"    │            │
│ — 김OO 교사 │ — 이OO 교사 │ — 박OO 교사│
└────────────┴────────────┴────────────┘
```

## 섹션 6: 최종 CTA (신규)

> 노션 구현: 콜아웃 블록 또는 배경색 + H3 + 버튼 링크

```
╔══════════════════════════════════════╗
║  지금 바로 노션톡 커뮤니티에 참여하세요  ║
║  11,000명의 선생님이 함께하고 있습니다  ║
║                                      ║
║  [카카오톡 커뮤니티 참여]  [인스타그램]   ║
╚══════════════════════════════════════╝
```

## 요약: 현재 vs 개선

| 현재 (2개 블록) | 개선안 (7개 섹션) |
|---|---|
| 히어로 + 푸터 | 히어로 → 통계 → 서비스 카드 → 기능 하이라이트 → 후기 → CTA → 푸터 |

## 노션 블록 매핑

| 섹션 | 노션 블록 |
|------|-----------|
| 히어로 | H1 + 텍스트 + 이미지 + 북마크(버튼) |
| 통계 | 3열 컬럼 + H2 + 텍스트 |
| 서비스 카드 | 구분선 + H2 + 2열 컬럼 x 2행 |
| 기능 하이라이트 | 2열 컬럼 (이미지+텍스트 교차) |
| 사용 후기 | H2 + 3열 컬럼 + 인용 블록 |
| 최종 CTA | 콜아웃 + H3 + 북마크(버튼) |
| 푸터 | 텍스트 + 링크 |
