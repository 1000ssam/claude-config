추상 SW 개념(노션 블록·DB·수식 등)을 Unsplash 워크스페이스 사진 키워드로 매칭하는 시도는 사실상 동작하지 않는다. 사용자도 무드보드 단계에서 직관적으로 깨닫고 "대충 체크했다"고 인정함.

**Why:**
1. 의미 격차 — 추상 SW 개념과 물리 사진 사이에 실제 의미 다리가 없음. 영문 키워드로 번역하는 단계에서 이미 손실
2. 워크스페이스 사진은 시각적으로 호환됨 — 어느 사진이든 비슷한 분위기. 클러스터링이 사실상 무의미
3. 갤러리 뷰의 목적은 콘텐츠 식별이 아닌 그리드 미감 — 유튜브 썸네일과 다름

**How to apply:**
유사 요청("블로그/DB 페이지에 어울리는 사진 자동 매칭") 받으면, 키워드 매칭으로 시간 쓰기 전에 다음 대안을 먼저 제시:
- **큐레이션 뱅크**: 사용자가 30~50장 직접 ✓ → 단일 풀 라운드로빈 (톤 통일 최강, 매칭 포기)
- **컬러 필터**: Unsplash `color=`로 톤 필터링 후 라운드로빈 (팔레트 정합)
- **AI 생성**: 진짜 의미 매칭이 가치라면 Midjourney/Sora로 클러스터별 일관 일러스트

가장 흔한 정답은 **큐레이션 뱅크 + 라운드로빈**. 매칭이 진짜 가치 있는 케이스(콘텐츠 썸네일·캠페인 비주얼 등)에만 키워드/AI 매칭 추천.
