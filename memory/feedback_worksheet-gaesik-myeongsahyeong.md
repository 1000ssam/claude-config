학습지/워크시트(특히 [[project_eastasia-worksheet]]) 본문은 **개조식 + 명사형(체언/음슴체) 어미**로 쓴다. 서술형 완결 문장(`~하였다`, `~번성하였다`)으로 쓰면 안 됨(사용자 강하게 거부).

**형식 (기존 동아시아사 페이지 06·07 기준):**
- 헤딩: `# 1. 대주제` / `## 1) 소절` / `### 항목`
- 본문: `bulleted_list_item`로, **핵심어 볼드 inline** + `→` 화살표로 인과·순서 압축
- 예: `- 명, 건국 초부터 **해금 정책** 시행 → 조공 무역만 허용, 사무역 통제`
- 종결: 명사·체언 마무리(`통제`, `강화`, `번창`, `쇠퇴`) — 완결 문장 금지
- 자료/용어: `callout`(gray_bg)로 분리

**Why:** 수업 슬라이드·학습지로 바로 쓰는 포맷. 줄글은 가독성·판서용으로 부적합.

**How to apply:** 근본 해결은 스킬에 규칙을 박아두는 것 — eastasia-worksheet STEP 2 ‘본문 작성 형식’에 개조식·명사형 규칙을 명시했고 ❌목록에 줄글 금지도 추가함(2026-05-26 보완 완료). 매번 기존 페이지를 대조하는 건 과설계이므로 하지 않는다(사용자 지적). 스킬 규칙을 신뢰하고, 정말 새로운 포맷이거나 모호할 때만 기존 산출물 참고. 관련: [[feedback_textbook-headings-verbatim]].
